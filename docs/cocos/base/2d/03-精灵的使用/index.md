# 03-精灵的使用

精灵(图片)，单色(颜色)

## 精灵

来学习下精灵节点

### type 渲染模式

- Simple 普通

- Sliced 九宫格 应用场景 聊天气泡等

- Tiled 平铺  应用场景 背景地板平铺等

- Filled 填充 应用场景 血条 进度条等

### Size Mode 所在节点的尺寸

指定 Sprite 的尺寸

- Trimmed 表示会使用原始图片资源裁剪透明像素后的尺寸
- Raw 表示会使用原始图片未经裁剪的尺寸
- Custom 表示会使用自定义尺寸。当用户手动修改过 Size 属性后，Size Mode 会被自动设置为 Custom，除非再次指定为前两种尺寸。
