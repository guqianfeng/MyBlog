# 频繁记录自己，能改变什么？

频繁记录自己，真的能改变拖延，提升效率，撬动人生嘛？

答案应该是：是也不是

记录这个动作关键不是记，而是记什么，为什么要记

## 拖延日记

核心原理：将无意识的惯性行为 -> 有意识的自我觉察

之前看到过一个高赞的帖子，作者通过记录拖延日记改善了自己的拖延症

“当我开始记录自己的拖延，我的生产力暴增”

只要自己开始拖延就会立即记下：我本该做什么，我正在做的事，我当时的感受

在一天结束后，去回看这些笔记，在最开始的时候，这些记录毫无意义，但慢慢地，作者就发现了这些记录里的一些重复的模式

原本以为拖延只是懒，但后面发现，每当拖延的时候，是因为不知道从哪里开始，这种不知所措的感觉都和某种特定类型的任务有关！有了这个意识，拖延就可以容易应对！
有意识的记录，把无意识的拖延变成了，可以观察可以被分析的信息，观察自己的思维和行为模式

这种观察就可以显著的减少自动化的逃避反应，就不会归因于懒，而是更好地去解决问题

## 情绪日记

记录情绪事件，记录感受，定期复盘

比如，我现在感觉有点烦躁！这种记录就毫无意义

今天会议之后领导模糊的反馈，让我感受到有一些不被认可，和前途未卜的焦虑！

这种记录反而会让你平静下来，客观地看待问题，知道是什么导致了情绪，找到问题的核心

## 食物日记

情绪性进食的恶性循环不知道大家有没有经历过。反正我是经历过的。上班加班累了就想去吃顿好的犒劳自己

压力大了点奶茶，孤独时吃零食，焦虑中暴食甜品，食物成为安抚情绪的创口贴，但贴完后又陷入更深的失控感和自责！用食物应对情绪是无意识自动化的！接下去你可以记录这样的食物笔记

记录触发情境，标注当时的身心信号，观察行为模式，追踪后续的反应

你并不是因为馋，而是用食物去应对未被处理的情绪，

## 晨间/晚间日记

早晨一天的黄金时间，记录今天对于我来说最重要的三件事是什么

写出目标能大大的增加目标实现的概率，你潜意识会自动锚定那个重要的点

早上的记录可以排除很多其他的干扰

睡前5分钟记录今天最后收获的三件事，以及一件对于明天来说相对重要的事
不一定要和早晨的一一对应，毕竟会有突发情况发生
晚间的记录重点是放下焦虑，认可自己，以更好的姿态迎接明天

如果目标过于繁重，你也可以把目标换成体验，比如晨间日记，记录你今天三个最想要的体验
然后围绕这三个体验去设计当天的生活

比如今天想和朋友分享之前的一个喜悦，那就找机会和你朋友聊天
想拥有事情做完的成就感，就可以带着期待去完成这些事情，
比如想吃什么美食，就提前规划好美食计划等等

这样就可以把生活打造成你所期待的样子


## “无效”日记

如果以上日记还是无效，那可能是因为缺少一个核心元素，那就是被看见
被看见在一个人的心理发展中，发挥着重要作用，他可以帮你确认你的感受，验证你的存在，获得外界的支持！

是否可以坦诚面对自己重复的某种模式
是否可以放下偏见去看到这些模式背后的一些需要
是否有勇气去面对内心深处的另一个自己

记录之后的这些，看见，反思，察觉，整合才会让记录变得有效有意义
